# Cloud Foundation – AWS Web Environment

Projeto prático desenvolvido durante a formação em Cloud Foundation, com o objetivo de aplicar conceitos fundamentais de infraestrutura em nuvem utilizando serviços da AWS.

Este projeto reúne as evidências e entregas relacionadas ao provisionamento, publicação e monitoramento de uma aplicação em ambiente cloud, servindo como registro da minha evolução técnica durante a transição para a área de Cloud Computing.

## Objetivos

* Provisionar infraestrutura utilizando serviços AWS;
* Aplicar conceitos de alta disponibilidade e monitoramento;
* Configurar serviços de rede e resolução de nomes;
* Publicar aplicações em ambiente cloud;
* Consolidar conhecimentos adquiridos por meio de atividades práticas.

## Serviços AWS Utilizados

* Amazon EC2
* Amazon S3
* AWS IAM
* Amazon Route 53
* Amazon CloudWatch
* Elastic IP

## Atividades Desenvolvidas

* Provisionamento e gerenciamento de múltiplas instâncias EC2;
* Criação e reutilização de imagens através de AMIs;
* Configuração de Launch Templates;
* Configuração de domínio e registros DNS utilizando Route 53;
* Implementação de Health Checks para verificação de disponibilidade;
* Publicação de aplicação estática em bucket S3;
* Configuração de monitoramento utilizando CloudWatch;
* Validação do ambiente através do acesso à aplicação via navegador.

## Principais Aprendizados

Durante o desenvolvimento deste projeto, aprofundei conhecimentos relacionados a:

* Provisionamento de infraestrutura em nuvem;
* Conceitos de DNS e resolução de nomes;
* Monitoramento e disponibilidade de serviços;
* Reutilização e padronização de ambientes;
* Publicação de aplicações utilizando serviços gerenciados da AWS.

## Observações

Este projeto foi desenvolvido em ambiente acadêmico com fins educacionais, representando uma aplicação prática dos conceitos estudados e servindo como evidência da minha jornada de transição para a área de Cloud Computing.
