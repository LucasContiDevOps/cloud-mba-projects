# cloud-mba-projects

Projetos práticos desenvolvidos ao longo do MBA em Cloud Computing & DevOps da Impacta.

Este repositório reúne laboratórios, entregas e experimentos realizados durante a formação, servindo como um registro da minha evolução técnica na área de Infraestrutura, Cloud e DevOps.

##Tecnologias e conceitos abordados

☁️ AWS
🌐 Redes
🐧 Linux
🐳 Containers
🏗️ Cloud Architecture
⚙️ Cloud DevOps e SRE
🔄 Integração entre serviços
🚀 Criação de pipelines (CI/CD)
📉 Redução de Toil por meio de automação

##Objetivos deste repositório

Documentar projetos e atividades desenvolvidas durante o MBA;
Consolidar conhecimentos adquiridos ao longo da formação;
Acompanhar minha evolução técnica;
Compartilhar evidências práticas da minha transição para a área de Cloud e DevOps.

##Estrutura

Cada diretório contém a documentação, evidências e materiais relacionados às atividades desenvolvidas, incluindo projetos práticos, pipelines, automações e estudos aplicados.

##Observação

Este repositório não tem como objetivo apresentar soluções definitivas ou arquiteturas de produção, mas sim registrar minha jornada de aprendizado e aplicação prática dos conceitos estudados.
